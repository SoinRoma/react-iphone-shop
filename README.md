### Запуск проекта для Windows

1. Запустить проект в IDE (WebStorm)

2. Установить все зависимости(IDE сам предложит).
```
npm install
```
3. Запуск проекта
```
npm start
```

4. Деплой приложения в Github Pages
```
npm run deploy
```
